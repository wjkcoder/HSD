HSD 是汉得智能制造事业部的智能制造研究室设计制造的设备云驱动产品，团队拥有资历深厚的设备控制器工程师，熟练的自动化工程师，高级程序架构师，高级算法设计程序人员，数据库设计师，高质量运维团队。


HSD 致力于打造纯中国制造的设备数据采集应用与控制一体化驱动云，不依赖opc任何第三方软件，拥有自行编写庞大的数据通路底层协议集群，几乎覆盖市场上所有主流品牌系列，非标设备可扩展，实现真正中国制造2025。

汉得合作伙伴

福耀集团是全球范围内汽车玻璃生产巨头，是国家计划中国制造2025首批重点实验项目。汉得信息携手福耀以高质量，高标准实现设备的数字化和智能化，打造中国制造智能工厂。福耀玻璃项目一期从福清总部开始，逐渐扩散到二期的重庆，长春，武汉等全国所有事业部，获得广大认可。



贸联集团经营全球范围内的网络连接器材，是国际连接解决方案的领导供货商。汉得信息智能制造团队与贸联电子合作，实现贸联电子设备的智能化集成，加强贸联电子在设备信息化，智能化方面上高效管理。
